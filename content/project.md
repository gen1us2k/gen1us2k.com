Все проекты доступны на [Github](https://github.com/gen1us2k) . В основном это всякие мелкие пет-прожекты.
Наиболее интересные проекты:

1. [MeshBird](https://github.com/meshbird/meshbird) - overlay сеть для чего угодно под Linux/FreeBSD/MacOs
2. [Ariadna](https://gen1us2k.com/gen1us2k/ariadna) - геокодер, который отлично ищет адреса в Бишкеке. Демок открытых нет :(
3. [Python-whois](https://github.com/gen1us2k/python-whois) - либа для питона, для проверки статуса домена.
4. [Django-OSMP](https://github.com/gen1us2k/django-osmp) - прием платежей по OSMP протоколу для Django. По хорошему нужно переписать его и написать тесты.
